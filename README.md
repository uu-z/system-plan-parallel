# system-plan-parallel

High-leverage Codex skill for holistic planning, bounded parallel execution, and quantified decision-making.

## Install

Copy or sync this folder into your Codex skills directory:

```bash
mkdir -p "${CODEX_HOME:-$HOME/.codex}/skills"
rsync -a /path/to/system-plan-parallel/ "${CODEX_HOME:-$HOME/.codex}/skills/system-plan-parallel/"
```

## Use

Invoke it explicitly:

```text
Use $system-plan-parallel to score this task, produce a full-system plan, and split it into safe independent workstreams.
```

Short version:

```text
Use $system-plan-parallel. Keep it concise.
```

Typical triggers:

- "Think holistically before coding."
- "Give me the overall plan first."
- "Break this into safe parallel workstreams."
- "先整体分析，再并行实施。"
